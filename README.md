# Модель прогнозирования цен на жилье (California Housing)

Этот проект представляет собой систему машинного обучения, которая предсказывает медианную стоимость недвижимости в округах Калифорнии на основе данных переписи населения.

## Описание проекта
Модель анализирует 8 ключевых признаков, таких как:
* Средний доход в районе
* Средний возраст домов
* Количество комнат и спален
* Популяция и количество домохозяйств
* Географическое положение (широта и долгота)

## Технологический стек
Проект написан на **Python 3.x** с использованием следующих библиотек:
* **Pandas** — для обработки и анализа табличных данных.
* **Scikit-learn** — для построения и обучения модели машинного обучения.
* **Random Forest Regressor** — основной алгоритм (случайный лес), обеспечивающий высокую точность.
* **Joblib** — для сохранения и загрузки обученной модели.

## Структура репозитория
* `data/` — папка, в которую автоматически загружается датасет в формате CSV.
* `models/` — папка для хранения файла обученной модели (`.pkl`).
* `main.py` — основной файл проекта: загрузка данных, обучение модели и оценка результатов.
* `requirements.txt` — список библиотек, необходимых для работы проекта.
* `.gitignore` — файл, скрывающий лишние системные файлы от Git.
